### Object Oriented Programming App 8: Automated Emails

- **Description**: This app reads users' names, email addresses, 
and interests from an Excel file and sends an email at a certain time every day to each user with news 
feeds about the users' interest. API (`newsapi.org`) is used for 
extracting news titles and the related links.

- **Structure**: `design.txt` includes the class structure of the code
and `requirements.txt` includes the packages to be installed 
for this project.
